Github Projects

SimpleSpy

A simple, requirement free, scroll spy. SimpleSpy uses custom data attributes to track which elements to watch and which to modify. SimpleSpy also allows you to run functions within the scope of the watched and the modified elements.

Project Site

Gridl

Gridl is an attempt to bridge the gap between current layout practices and new grid based practices. It supports Bootstrap style Containers, Flex Containers and columns with grid. This however does mean that it does not correctly render in legacy browsers at this time and polyfills cause major performance issues.
