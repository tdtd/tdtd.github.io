Why Grid?

  • Grid is easy to use. CSS Grid operates in a way that is immediately comprehensible. You can choose exactly which regions of the screen in which you would like your element to be placed.
  • Grid is easy to understand. Grid works how designers think. Instead of floating div shifting around the page, each element can be specifically placed.

Why Gridl?

  • Gridl is a combination of current responsive syntax and ideas with new CSS Grid ones.
  • Gridl is designed to allow for more flowy grid sizes to move around set grid elements.

Why You Shouldn't Use Grid.

  • CSS Grid is not supported across all browsers.
  • When CSS Grid is implemented it is not always done so in a consistent manner.

Responsive Media

Responsive Images

Use the .res-img class to make an image responsive.

Responsive Video

Place a video tag inside of div with the .video class to make a responsive video.

Responsive Embedded Video

Place an iframe for an embedded video inside of div with the .embed class to make a responsive iframe.

Responsive Embedded Audio

Give an <audio> tag the .res-audio class to make a responsive iframe.
